# LiturgyBridge

## Open-source multilingual digital companion for Orthodox worship

LiturgyBridge is an open-source project that helps people follow and understand Orthodox worship across language barriers.

The project combines structured liturgical texts, multilingual presentation, synchronization technologies, and optional AI-assisted translation tools.

## Vision

Orthodox worship has been preserved for centuries through rich liturgical traditions and languages such as Church Slavonic, Greek, Arabic, and others.

At the same time, many communities today are multilingual. Visitors, younger generations, and people from different cultural backgrounds often struggle to fully understand the services.

LiturgyBridge aims to build a bridge between tradition and accessibility.

---

## Core principles

- The liturgical tradition remains unchanged.
- Verified liturgical texts have priority over automated translations.
- Technology supports understanding and participation.
- Communities can contribute translations and improvements.
- Open standards and open collaboration are encouraged.

---

## Planned features

- 📖 Multilingual liturgical text display
- 🔄 Synchronization with the current service section
- 🌍 Multiple language support
- 📱 Browser-based access via QR code
- 🎧 Optional audio assistance
- 🤖 AI-supported translation for non-fixed speech
- 🏛️ Support for different Orthodox traditions
